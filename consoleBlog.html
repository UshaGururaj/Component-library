The most frequently used methods are logging of text and other data. They are console.error(), console.info(), console.log(), console.warn() and we will discuss about all these frequently used methods.

  • console.log()

    • This method outputs the message to the web console. That message may be string, integer or float numbers, array, objects.

    Syntax :console.log(message);

    This function takes message as a parameter. This message can be an object(s), string, or array data type.

    Example to write string to the console.

    console.log("printing on console");

    console.log("Student name is " + studentObj.name + ".His age is " + studentObj.age +" and he secured " + studentObj.gpa + " GPA.");

  • console.dir()

    • This method lists all properties of javascript object which we pass as a parameter.

    Syntax :console.dir(object);

    Then what is the difference between console.log() and console.dir()?

    When we try to console DOM elements in chrome there we can see useful difference.That is, console.log() prints the element in a HTML - like tree.But, console.dir() prints the element in a JSON - like tree.

  • console.error()

    • This method help us to output errror message to the web console.

    Syntax :console.error(message);

  • console.info()

    • This method outputs an informational message to the web console.

    Syntax :console.info(message);

  • console.warn()

    • This method outputs a warning message to the web console.

    Syntax :console.warn(message);
