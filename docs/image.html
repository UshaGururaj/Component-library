Image

Responsive images are images that never becomes wider than their parent container.

Responsive Image

Images can be made responsive by adding img-responsive class to the <img> element.

Round Image

Round or circular images are used in lot of place, such as profile picture. You can make image round or circular using img-round class which apply border-radius: 50% to the <img> element.
