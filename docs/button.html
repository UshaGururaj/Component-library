Button

Buttons are one of the most common UI elements. They make it possible for users to interact with a system and take action by making selections. They are also called call to action (CTA). Buttons that you can use in your website's forms, website homepages, dialog boxes, toolbars and more.

Simple Button

Castor-UI provides different types of buttons such as Primary, Secondary, Success, Danger, Warning, Info etc. Use btn class with respective contextual classes btn-primary , btn-secondary , btn-success , btn-danger , btn-warning , btn-info to implement buttons in your website.

Outline Button

There are two types of outline buttons, Primary and Secondary. Use btn class with respective contextual classes btn-primary-outline , btn-secondary-outline

Disabled Button

Make buttons inactive by adding the disabled boolean attribute to any <button> element. A disabled button is unusable and un-clickable.

Button with different sizes

Use btn-l and btn-s class to implement large and small buttons in your website.

Button Link and Icon

You can use icons as a button which are fully accessibile to screen readers and also link as a button. Use contextual classes btn-link and icon-btn-primary to implement these buttons.

Toggle Button

A toggle switch is a digital ON/OFF switch. It allows the user to change a setting between two states. Toggles may replace two radio buttons or a single checkbox to allow users to choose between two opposing states.
