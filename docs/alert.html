Alert

Alerts are used to get user's attention for important information without interrupting the user's flow.

Simple Alerts with Icon

Use class="alert" class with respective contextual classes to give user different messages based on context such as success, error, or warning, etc. Those contextual classes are class="alert-success" , class="alert-warning" , class="alert-error" , class="alert-info" .
