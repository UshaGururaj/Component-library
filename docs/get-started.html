Introduction

Welcome! Get started with Castor-UI and build responsive web pages in no time. You just need to add our stylesheet to your HTML page and copy-paste the HTML code given and customize your content.

Quick Start

Looking to quickly add Castor-UI to your project? Copy-paste the stylesheet <link> into your <head> element before all other stylesheets to load our CSS.
