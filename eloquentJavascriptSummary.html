Values, Types and Operators

Values : In Javascript, chunks of bits which represent pieces of information are called values.

Types : Values are madeup of bits and each value plays different role. This role is determined by the value type.

Ex: Number, Text / String, Function.

To create a value, we must name it ans these value has to be stored some where. It will store with that name.

Basic Data types

  1. Number : It contains integer, float numeric values.

    2. Ex : 57, -21, 20.34..

  2. String : Strings are used to represent text or group of characters. Strings always enclosed within quotes.

    3. Ex : `Down on the sea`, "Lie on the ocean", 'Float on the ocean'..

    We can use single quote, double quote or backtick to represent strings, but both start and end quotes should match.

Backslash(\) is called as escaping character. The character after backslah will has a special meaning. A quote preceded by a backslash will not end the string but it will be part of it.
